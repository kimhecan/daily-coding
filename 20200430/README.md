### 4. 반성

조건에 n + m >= k + 3을 추가하면.. 아래 코드가 필요없어진다. +3은 n 은 2보다 m은 1보다 커야 팀이 많들어지기때문에 들어가 있는거고 k가 찌꺼기라고 생각할때 n+m의 지꺼기가 k 찌꺼기보다 커야 team을 추가할 수 있다는 의미이다.


아니면 편하게 의식의 흐름대로 풀어도 된다.. 문제 없잖아?
