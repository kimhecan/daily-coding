### 1. 이해


최대 몇개의 회의가 하나의 회의실에서 열릴 수 있는지 알아내는 문제다. 하나의 회의가 총 몇개인지 입력받고 각각

에 대하여 시작시간과 종료시간을 입력받는다. 시간이 겹치면 안되고 끝나는 시간과 시작시간이 동일할 수 있다.


### 2. 계획
